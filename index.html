Freespeak

"We listen to your needs, not your calls"

Private Voice Communication

An open-source, real-time voice chat application built with privacy at its core. Host your own server and communicate freely.

View on GitHub Download

Features

Real-Time Voice Chat

Low-latency voice communication with multiple participants in rooms. Server-side audio mixing ensures smooth conversations.

Privacy First

Self-host your own server. Your conversations stay on your infrastructure, not on third-party services.

Text Chat

Send messages alongside voice. Typing indicators let others know when you're composing a message.

Cross-Platform

Native applications for macOS, Linux, and Windows. Same great experience everywhere.

Open Source

BSD-3-Clause licensed. Inspect the code, contribute, and customize to your needs.

Simple Setup

Easy to deploy server and intuitive client interface. Get your team talking in minutes.

Built With

  • Go
  • Fyne (GUI)
  • PortAudio
  • WebSockets
  • UDP Streaming
